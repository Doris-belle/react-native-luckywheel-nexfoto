# react-native-spin-the-wheel

A react native package for spin the wheel game

## Installation

```sh
npm install react-native-spin-the-wheel
```

## Usage

```js
import { multiply } from 'react-native-spin-the-wheel';

// ...

const result = await multiply(3, 7);
```
